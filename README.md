## 2. 수업에 대한 기대 / 포부

한 학기 동안 **'응용소프트웨어개발'** 과정을 성실히 이수한 미래의 나에게 이 편지를 보낸다.

현재는 코딩과 소프트웨어 설계의 세계가 다소 낯설고, 실제 작동하는 프로그램을 기획하는 과정에 많은 고민이 있다. 하지만 학기가 끝날 무렵의 나는 지금보다 훨씬 성장한 개발자의 시각을 갖게 되기를 기대한다.

이번 수업을 통해 단순한 코드 작성에 그치지 않고 **응용 소프트웨어 개발 전반에 대한 이해**를 얻고, API 연동과 시스템 설계 등 기술적인 부분에서도 의미 있는 발전을 이루기를 바란다.

---

## 3. 수업을 통해 만들어보고 싶은 소프트웨어

이번 학기 프로젝트를 통해 **개인 학습 진도 및 집중력 관리 도구**를 개발하고 싶다.

사용자가 학습 목표를 설정하고 학습 시간과 집중도를 기록하여 관리할 수 있는 시스템을 만들어 학습 효율을 높이는 소프트웨어를 구현하는 것이 목표이다.

---

## 4. 수강 신청 이유

응용 소프트웨어를 실제 프로젝트 형태로 배우기 위해 이 강의를 수강하였다.

특히 Python 프로그래밍 능력을 향상시키고 다양한 개발 도구를 활용하는 경험을 쌓고 싶다. 또한 실제 소프트웨어의 **Deployment 과정**을 경험하여 Smart Farm 분야에서 활용할 수 있는 지능형 시스템 개발 능력을 기르고자 한다.
